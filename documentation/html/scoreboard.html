Scoreboard Module¶

class scoreboard.Scoreboard¶

Bases: object

Scoreboard class for use in battleship. will print only values passed

Example:

scoreboard = Scoreboard()
scoreboard.printScoreboard(_)
printScoreboard(is_cpu, p1_ships, p1_hits, p1_shots, p1_games, p2_ships, p2_hits, p2_shots, p2_games)¶

prints the whole scoreboard, calls functions as needed

Parameters

  • is_cpu (bool) – true if cpu player

  • p1_ships (int) – player 1 Remaining ships

  • p1_hits (int) – player 1 total hits

  • p1_shots (int) – player 1 total shots taken

  • p1_games (int) – player 1 total games won

  • p2_ships (int) – player 2 Remaining ships

  • p2_hits (int) – player 2 total hits

  • p2_shots (int) – player 2 total shots taken

  • p2_games (int) – player 2 total games won
